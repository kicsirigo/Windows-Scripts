What this script do?
This script provides a graphical interface for users to select and compare two text files. (FIRST file is the bigger that you want to compare against, the SECOND file is the smaller file, the one you want to compare)

Usage:

1. Open PowerShell as a regular user
2. cd into the folder where this .ps1 file is located
3. Run the Set-ExecutionPolicy -ExecutionPolicy Unrestricted -Scope Process -Force command in PowerShell
4. Run the script.
5. Use the graphical file picker to select the first .txt file, then the second .txt file
6. Use the folder browser dialog to select the location where you want to save your report

Example usage: You need to compare printer list form 2 printservers.
